# Make WordPress Training – Office Hours

This repository supports the monthly **Office Hours** meetings of the [Make WordPress Training](https://make.wordpress.org/training/) team.

## What are the Office Hours?

A monthly 60-minute video meeting focused on actionable discussion, concrete task assignment, and onboarding new contributors. Read the full proposal on the Make WordPress Training blog:

→ [Proposal: Monthly Office Hour Meeting for Make WordPress Training](https://make.wordpress.org/training/2026/02/03/proposal-monthly-office-hour-meeting-for-make-wordpress-training/)

## How the Notes Workflow Works

1. Note-taker creates a draft in `/notes/` during or after the meeting
2. Participants have ~48 hours to comment and suggest corrections via GitHub
3. Once approved, the notes are published as a blog post on [make.wordpress.org/training](https://make.wordpress.org/training/)
4. The GitHub file links to the published blog post
