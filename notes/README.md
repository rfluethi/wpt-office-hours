# Meeting Notes

This folder contains working drafts of Office Hours meeting notes.

## Workflow

1. The note-taker creates a file here **during or shortly after** the meeting
2. Participants have **~48 hours** to suggest corrections by commenting on the file or [opening a Document Correction issue](https://github.com/rfluethi/wpt-office-hours/issues/new?template=document-correction.md)
3. After the correction window closes, the moderator reviews any feedback and publishes the notes as a blog post on [make.wordpress.org/training](https://make.wordpress.org/training/). If no corrections are received, the notes are considered approved and published as-is.
4. The file is then updated with a link to the published blog post

## Filename Convention

```
YYYY-MM-DD-meeting.md    ← meeting notes
YYYY-MM-DD-agenda.md     ← agenda (published before the meeting)
```

Example: `2026-04-19-meeting.md`, `2026-04-19-agenda.md`

## Template

Use [`/templates/meeting-notes.md`](../templates/meeting-notes.md) for notes and [`/templates/agenda.md`](../templates/agenda.md) for agendas.

## Published Notes

All approved notes are published on the [Make WordPress Training blog](https://make.wordpress.org/training/). Links are added to the files here once published.
